How fast are humans going?

I was curious about the fastest a human had gone each year. My initial thought was that humans traveled around the speed of a horse galloping for centuries. But when did bicycles or motorized technology take off? And when did flying and then space flight expand?

I couldn’t find a good dataset, so Claude provided one and Gemini double-checked it. I was more interested in the visualization than the data scraping.

df_milestones = pd.DataFrame(milestones, columns=["year", "speed_mph", "mode", "note"])

# Create year-by-year from -6000 to 2026, forward-fill
all_years = pd.DataFrame({"year": range(-6000, 2027)})
df_full = all_years.merge(df_milestones, on="year", how="left")
df_full[["speed_mph", "mode", "note"]] = df_full[["speed_mph", "mode", "note"]].ffill()

# Derived columns
df_full["speed_kmh"] = round(df_full["speed_mph"] * 1.60934, 1)

df_full["year_label"] = df_full["year"].apply(
   lambda y: f"{abs(y)} BCE" if y < 0 else (f"{y} CE" if y > 0 else "1 BCE")
)

def assign_era(row):
   mode = row["mode"]
   if mode == "Running":                           return "Human Power"
   if mode in ("Horseback", "Chariot"):            return "Animal Power"
   if mode in ("Steam Locomotive", "Steam Car"):   return "Steam Power"
   if mode == "Automobile":                        return "Automobile"
   if mode == "Aircraft":                          return "Aviation"
   if mode == "Jet Aircraft":                      return "Jet Age"
   if mode == "Rocket Plane":                      return "Rocket / Jet"
   if mode == "Spacecraft":                        return "Space"
   return "Other"

df_full["era"] = df_full.apply(assign_era, axis=1)
df_full["is_milestone"] = df_full["year"].isin(df_milestones["year"])
